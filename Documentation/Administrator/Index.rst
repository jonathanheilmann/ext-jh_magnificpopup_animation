Administrator Manual
====================


.. _admin-installation:

Installation
------------


#. Go to the Extension Manager
#. Install the extension
#. Load the static template "Magnific Popup - Animation Effects (jh_magnificpopup_animation)"


.. _admin-configuration:

Configuration
-------------

To add an animation to the image Lightbox

#. go to Constant Editor
#. select category "PLUGIN.TY_JHMAGNIFICPOPUP"
#. add animation-class to constant "mainClass"
#. add removalDelay to constant "removalDelay"

For a list of animation-classes and removalDelays please see :ref:`available-animations`
