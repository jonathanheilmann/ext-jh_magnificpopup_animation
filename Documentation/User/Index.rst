Users Manual
============

**Animations for images** are set globally and may not changed for single pages. Please see chapter :ref:`admin-manual`

| **Animations for plugins** may be set globally and locally, whereas the local setting overrides the global setting.
| To use an locally set animation, select an animation for "mainClass" and the same one for "removalDelay".
| For a list of available animations please see :ref:`available-animations`
